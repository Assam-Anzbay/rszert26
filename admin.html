Sign in

Don't have one yet? On GitHub go to Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token. Under "Repository access" choose only this repository, and under "Permissions" set Contents: Read and write. Nothing else is needed. Paste the token above — it's kept only in this browser tab and is never saved anywhere in the site's code.
